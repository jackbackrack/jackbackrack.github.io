Feedback Driven Airbrush Machine
What Controlling a CNC airbrush machine with video feedback.
Which The set up is a camera pointed at the paint surface along with a CNC airbrush machine and software to close the feedback loop.
Why I'm really interested in feedback in general. It is pervasive in nature, in particular, in growth of plants and animals. I love the loose style of sloppy paint and errant spray and want to guide it using an outer feedback loop modeling like what humans do when they use error prone techniques in order to achieve a pleasing result.
Who Jonathan Bachrach along with Bill Tremblay who built the CNC airbrush machine itself
How Sony camera, CNC airbrush machine, custom software
